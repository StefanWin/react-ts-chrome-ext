## Setup
- Update `package.json` with your metadata
- Add your own icons with [favicon.io](https://favicon.io/)
- `npm i`
- Update `src/manifest.json` with your metadata and [required permissions](https://developer.chrome.com/docs/extensions/reference/)
- [Enable developer mode in Chrome](https://developer.chrome.com/docs/extensions/mv3/getstarted/#manifest)
- `npm run build`
- [Use the `dist` folder for Load Unpacked](https://developer.chrome.com/docs/extensions/mv3/getstarted/#manifest)
